This folder contains two subfolders

1. The 'Localization' folder contains the localization program we implemented

2. The 'Haar cascade classifiers' folder contains all the datasets used for training the classifier and the models we get from training.

3. The 'Classification' folder contains Section 2 as well as the last part of Section 3.1 (classification+localization). 

4. The training images folder 'Train' is assumed to be in the 'Classification' folder (same directory as the classification jupyter notebooks)
